# Synthetic-mobile-sales-2025
EDA project analyzing using mobile sales data 2025.
Mobile Sales Data Analysis 📱
This project focuses on performing Exploratory Data Analysis (EDA) on a mobile sales dataset containing 500+ records. The analysis aims to uncover meaningful insights related to product performance, pricing strategies, and customer purchasing behavior.

At first ,I downloaded a Mobile sales dataset from Kaggle and worked on it using libraries like Numpy ,Pandas ,Matplotlib and Seaborn in Google Colab to analyze the data and create meaningful visualizations.
In this project, I focused on understanding the dataset by performing data cleaning ,handling missing values ,exploring patterns and visualizing insights using python libraries.

The insights derived from this analysis can help businesses make data-driven decisions, optimize pricing strategies, and better understand market demand in the mobile sales domain.
